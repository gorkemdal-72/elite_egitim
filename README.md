# Elite Eğitim Projesi

Bu proje, Elite Eğitim web platformunun kaynak kodlarını içerir. Proje, Vite, React, TypeScript ve Tailwind CSS ile geliştirilmiştir.

## Projeyi Çalıştırma

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

### 1. Gereksinimler

- Bilgisayarınızda **Node.js** yüklü olmalıdır. [Buradan indirebilirsiniz](https://nodejs.org/).

### 2. Kurulum ve Çalıştırma

Terminalinizi (Komut İstemcisi veya PowerShell) projenin bulunduğu klasörde açın ve sırasıyla aşağıdaki komutları çalıştırın:

```sh
# Gerekli paketleri yüklemek için (Zaten yüklendiyse atlayabilirsiniz)
npm install

# Geliştirme sunucusunu başlatmak için
npm run dev
```

Sunucu başladıktan sonra, terminalde gösterilecek olan yerel bağlantıya (genellikle `http://localhost:8080` veya `http://localhost:5173`) tıklayarak siteyi tarayıcınızda görüntüleyebilirsiniz.

## Proje Hakkında

Bu proje tasarımı ve içeriği itibariyle Elite Eğitim vizyonuna uygun olarak düzenlenmiş olup, artık bağımsız bir proje olarak geliştirilmeye hazırdır. Herhangi bir yapay zeka oluşturucu aracına (Lovable vb.) bağlı değildir.
- Stiller için `Tailwind CSS`
- UI bileşenleri için `shadcn/ui` kullanmaktadır.
